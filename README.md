# Computational-Quantum-Chemistry-ORCA
# NPTEL: Quantum Computational Chemistry

This repository serves as a weekly log of my learning and computational exercises from NPTEL courses. It includes molecular modeling, quantum chemical calculations using **ORCA**, and visualizations from **Avogadro** and **Chemcraft**.

## 🛠 Tools Used
* **Avogadro_1.2.0:** Molecular building and geometry preparation.
* **ORCA_6.1:** Quantum chemistry engine for DFT and Ab-initio calculations.
* **Chemcraft_1.8:** Analysis of output files and visualization of molecular orbitals/spectra.

## 🧪 How to Use
Each week's folder contains the `.inp` files required to reproduce the calculations. To run a calculation:
`orca input_file.inp > output_file.out`

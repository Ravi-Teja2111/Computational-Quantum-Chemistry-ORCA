This folder contains ORCA calculations of NO(Nitric Oxide). It has input, output, BibTeX, and property files. 
To run ORCA on your device, 
1. Prepare the ORCA input file
2. Over terminal from that folder
3. orca input filename > output filename
4. It creates an ORCA output file. Don't abort the process. This may create temporary files during the process, but they will be deleted automatically.
5. Completed your ORCA calculation.

This folder contains ORCA calculations of FeO. It has input, output, bibtex, property files. 
To run ORCA in your device, 
1. Prepare the ORCA input file
2. Over terminal from that folder
3. orca input filename > output filename
4. It creates ORCA output file, Don't abrupt the process, It might create many temporary files through the process, But don't worry temporary files will be deleted automatically.
5. Completed your ORCA calculation.
